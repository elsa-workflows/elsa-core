# Elsa README Video

Remotion source for the short Elsa Workflows video used by the repository README.

## Commands

```bash
npm install
npm run studio
npm run render
npm run render:gif
npm run still
```

Generated assets are written to `../exports/`:

- `elsa-workflows-readme.mp4`
- `elsa-workflows-readme.gif`
- `elsa-workflows-readme-poster.png`
